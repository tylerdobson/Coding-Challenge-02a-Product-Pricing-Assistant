 # Product Pricing Assistant
  **Course:** ISM 3232 — Business Application Development
  **Assignment:** Coding Challenge 02a
  **Author:** Tyler Dobson

  A simple JavaScript program that acts as a pricing assistant for a small
  business. It tracks a product's cost, base price, discount, and sales tax,
  then calculates the profit per unit and the break-even quantity needed to
  cover fixed monthly costs.

  ## code

  - Declaring and using variables with primitive data types (string, number,
  boolean)
  - Building calculations with arithmetic expressions
  - Using `Math.ceil()` to round break-even units up to a whole number
  - Formatting currency output with `.toFixed(2)`
  - Printing clearly labeled results with `console.log()`

  ## sample when ran.

  Product Name: Neurogum
  Discounted Price: $16.99
  Final Price with Tax: $18.18
  Profit per Unit: $5.68
  Break-even Units: 793
  Is the product profitable? Yes
